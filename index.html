No Data Input Center

사용 방법:

  1. 구글 스프레드시트를 공개로 설정하세요 (링크가 있는 모든 사용자가 볼 수 있음)
  2. Google Apps Script를 사용하여 웹앱을 생성하세요
  3. 웹앱 URL을 입력하고 데이터를 입력하세요
  4. 미리보기를 확인한 후 제출하세요

주의사항:

  • 스프레드시트가 공개 설정되어 있어야 합니다
  • Google Apps Script 웹앱이 필요합니다
  • 브라우저 개발자 도구(F12)의 콘솔에서 자세한 오류 정보를 확인할 수 있습니다

홀덤 경기 관리

경기 정보 입력

형식: YYYYMMDD-번호 (예: 20250922-1, 20250922-2)
스프레드시트 1행을 검색하여 자동으로 다음 번호를 생성합니다.
💡 번호 생성 버튼을 누르지 않아도 전송 시 자동으로 생성됩니다!

참가자 정보

3칸 간격 셀 구조:
1번 경기: A1(인덱스) B1(경기번호) C1(딜러) → A2~(참가자)
2번 경기: D1(인덱스) E1(경기번호) F1(딜러) → D2~(참가자)
3번 경기: G1(인덱스) H1(경기번호) I1(딜러) → G2~(참가자)
• B2, C2, E2, F2... 빈 공간은 추가 정보용
참가횟수: 등수:

경기 정보 조회

조회 방법:
1. 날짜를 선택하세요
2. 해당 날짜의 경기 목록에서 조회할 경기를 선택하세요
3. 선택한 경기의 상세 정보를 확인하세요
날짜를 선택하고 "경기 목록 불러오기" 버튼을 클릭하세요
조회된 경기: 0개

경기 상세 정보

🏆 홀덤 리더보드

리더보드 기능:
1. leaderboard 시트에서 캐싱된 랭킹 정보를 불러옵니다
2. 참가자별 통계와 순위 정보를 표로 표시합니다
3. 총 경기, 승률, 순위 정보를 한눈에 확인할 수 있습니다
리더보드 불러오기: leaderboard 시트에서 랭킹 정보를 불러와서 표시합니다
리더보드 갱신: 모든 경기 데이터를 분석하여 leaderboard 시트를 새로 계산합니다

📋 대량 경기 입력

대량 입력 기능:
1. 달력을 통해 경기 날짜를 선택합니다
2. 딜러 정보와 참가자 목록을 텍스트로 입력합니다
3. 참가자명 뒤의 + 기호 개수로 참가 횟수를 자동 계산합니다
4. 한 번에 여러 경기를 빠르게 입력할 수 있습니다
선택하지 않으면 오늘 날짜가 자동으로 설정됩니다
첫 번째 줄에 "딜러: 이름" 형식으로 딜러 정보를 입력하세요 (선택사항)
딜러 정보가 없으면 바로 참가자 이름부터 입력하세요
+ 기호 개수로 참가 횟수를 표시합니다 (예: 닉+ = 2회, 에델++ = 3회)
순위권 참가자를 입력 순서대로 입력하세요 (선택사항)
첫 번째 줄은 1등, 두 번째 줄은 2등, 세 번째 줄은 3등으로 자동 할당됩니다
예: 호야
    라엘
    아우라
입력하지 않으면 모든 참가자가 "순위 없음"으로 처리됩니다
